No colors

Arrays in a 256 x 256 matrix

Binary format

Simplifies geometries

How does it work?

Mapbox Vector Tile Standard. Based on the binary protocal buffer (.pbf) from Google. Standard already used by Esri. github.com/mapbox/vector-tile-spec Web Mercator projection , Goolge tile scheme. GeoJSON mbtiles .pbf Data and Properties.
