What is a web map?

Data, tiles, styles and servers. Web map vs digital maps vs analogue maps View in a browser, scroll, pan, zoom.

History

How to provide this in a fast way? tiles!

Tiles

Tiles 256x256 pixels, same grid, same boundaries. All little tiles load way faster than one big map! Slippy maps.

Zoom levels

Each zoom level has its own set of tiles! Zoom level 0: 1 tile for the whole world. Increases exponentially... Zoom level 1: 4 tiles Zoom level 2: 16 tiles etc.

Serving tiles

Tiles rendered in advance. Stored in a cache. Tiles are just images.
